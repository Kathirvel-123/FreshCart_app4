# FreshCart – React Add-to-Cart Application

A simple and clean React-based shopping UI with an add-to-cart feature, built using Tailwind CSS and modular components.

## Features

* Add items to cart
* Prevent duplicate items
* Smooth modal for viewing cart
* Responsive UI
* Clean gradient navbar
* Fast and lightweight

---

## Project Structure

```
src/
├── App.jsx
├── main.jsx
├── index.css
├── components/
│   ├── Navbar.jsx
│   ├── CartModal.jsx
│   ├── ProductList.jsx
│   └── ProductCard.jsx
```

## Installation
### 1. Clone the repository
```

git clone ""

```

### 2. Install dependencies
```

npm install

```

### 3. Start the development server
```

npm run dev

````

## Tailwind css setup:

If you want the simplest setup, include this in `index.html`:
```html
<script src="https://cdn.tailwindcss.com"></script>
```
## Components Overview

## Navbar.jsx

Displays the app name and cart button with item count.

## ProductList.jsx

Shows sample products and handles add-to-cart actions.

## ProductCard.jsx

Individual product card component—displays image, title, price, and the "Add to Cart" button.

## CartModal.jsx

Beautiful modal with blur background and animated popup.
